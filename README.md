# git-github
Comandos básicos [GIT](https://git-scm.com/) e [GITHUB](https://github.com/)

## GIT CONFIG
```bash
git config --list
git config --global user.name "<NOME DO USUÁRIO>"
git config --global user.email "<EMAIL DO USUÁRIO>"
```
